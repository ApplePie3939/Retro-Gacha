# Retro-Gacha

# 🎰 古町レトロ・ガチャウォーク

新潟市古町エリアの商店街を舞台にした、店舗発見・回遊促進アプリのプロトタイプです。
ガチャで目的地を決め、実際に店舗を訪れてQRコードをスキャンすることでコインを貯め、お得なクーポンと交換できる体験を提供します。

## 📂 ディレクトリ構成

```
static/
├── css/
│   └── retro.css          # 全体共通のレトロデザインテーマ
├── images/
│   ├── qrcodes/           # 店舗ごとのQRコード画像
│   ├── shop_images/       # 店舗紹介用の画像
│   └── logo.png           # アプリロゴ
├── js/
│   ├── auth.js            # 認証処理（ログイン・新規登録）
│   ├── gacha.js           # ガチャ機能（API通信）
│   ├── qr.js              # QRコード読み取り機能
│   └── coupon.js          # クーポン機能
├── pages/
│   ├── index.html         # ログイン・新規登録画面
│   ├── gacha.html         # ガチャ画面
│   ├── qr.html            # QRコード読み取り画面
│   └── coupon.html        # クーポン一覧・交換画面
└── init_db.php            # データベース初期化スクリプト
```

## 🚀 導入・起動方法

1. データベースの初期化
   PHPがインストールされている環境で、以下のコマンドを実行してデータベースとサンプルデータを生成します。

Bash
php static/init_db.php

実行内容:

・SQLiteデータベース（gacha.db）の作成

・8つの実在店舗データと4つのクーポンデータの投入

・各店舗専用のQRコード画像（static/images/qrcodes/）の自動生成

2. ローカルサーバーの起動
   static/start_server.bat を実行するか、以下のコマンドでPHPのビルトインサーバーを起動します。

Bash
php -S 0.0.0.0:8080 -t static

3. ngrokによる外部公開（スマホ実機テスト用）
   店頭でのQR読み取りをシミュレーションするため、ngrokを使用して外部からアクセス可能なURLを発行します。

Bash
ngrok http 8080

発行された https://[ランダム文字列].ngrok-free.app をスマホのブラウザで開いて使用してください。

🎮 主な機能とサイクル1.ガチャを回す: ログイン後、ランダムに選ばれた

1店舗が表示されます。

2.店舗へ行く: 表示された住所や説明を元に、古町の街を探索します。

3.コイン獲得: 店舗に設置されたQRコードをスキャンすると、100コインを獲得できます（1店舗につき1回まで）。

4.クーポン交換: 貯まったコインを使用して、飲食店などの特典クーポン（大盛り無料、100円引き等）と交換できます。

## 🔌 技術スタック

Frontend: HTML5, CSS3 (レトロデザイン), JavaScript (Vanilla JS)

Backend: PHP 7.4+

Database: SQLite3

Tools: ngrok (外部公開), QR Code API (QR生成)

## ⚠️ 注意事項

本プロジェクトはプロトタイプ（デモ用）です。パスワードの暗号化などのセキュリティ対策は最小限となっています。

QRコードのスキャンは、URLパラメータとして shop_id を渡すことでシミュレーションされます。
